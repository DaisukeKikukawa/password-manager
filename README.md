# password-manager

## 使い方

```bash
# 任意の場所にクローン
git clone URL

# ダウンドードしたフォルダに移動し権限を付与
cd password_manager.sh
chomod +x password_manager.sh

# 以下のコマンドで実行 
 ./password_manager.sh 
```

## 機能

 - パスワードの登録
   - 以下の情報を保存
     - サービス名
     - ユーザー名
     - パスワード   
 - 登録済みパスワードの呼び出し
   - サービス名の入力が求められ、入力されたサービスのサービス名、ユーザー名、パスワードが表示される
